Interface to Add Face Data

Face Mask Detection Platform recognizes if a user is not wearing a mask. It can be connected to any existing or new IP mask detection cameras to detect people without a mask. If the camera captures an unrecognized face, a notification can be sent out to the violator.

Real-time Monitoring Dashboard

A user-friendly website allows the user to see who was not wearing a mask and see the photo or the video captured by the camera. User can also generate reports to download. the live camera feed can be view by the admin and the currents violaters can be seen

Notification center

If the face mask detector application identifies a user that he/she was not wearing a mask, sms alerts are sent with the picture of the person. It allows the application to run automatically and enforces the wearing of the mask.
